# Citation Style Guide

Guia de formato para citas in-text y lista de referencias en las tres normas principales. Selecciona la norma segun el contexto: APA para ciencias sociales/educacion, IEEE para ingenieria/ciencias de la computacion, Vancouver para ciencias de la salud/biomedicina.

---

## APA 7th Edition

### Configuracion del documento
| Elemento | Valor |
|----------|-------|
| Fuente | Times New Roman 12pt |
| Interlineado | Doble (2.0) |
| Margenes | 2.54 cm (1 in) en los 4 lados |
| Alineacion | Izquierda (no justificado) |
| Sangria primera linea | 1.27 cm (0.5 in) en cada parrafo |
| Numeracion de paginas | Superior derecha |
| Portada | Titulo, autores, institucion, curso, profesor, fecha |
| Referencias | Sangria francesa (1.27 cm), orden alfabetico |

### Citas in-text
```
(Author, Year, p. X)                        → parentetica
Author (Year) afirma que...                  → narrativa
(Author, Year, pp. X-X)                      → rango de paginas
(Author & Author, Year)                      → 2 autores (usar &)
(Author et al., Year)                        → 3+ autores
("Titulo Corto", Year)                       → sin autor
```

### Referencias
```
Articulo journal:
  Author, A. A., & Author, B. B. (Year). Title. Journal Name, volume(issue), pages. https://doi.org/XXXX

Libro:
  Author, A. A. (Year). Title (ed.). Publisher. https://doi.org/XXXX

Capitulo de libro:
  Author, A. A. (Year). Chapter title. In Editor (Ed.), Book title (pp. X-X). Publisher.

Conference:
  Author, A. A. (Year). Title. Proceedings of Conference Name, pages. https://doi.org/XXXX

Tesis:
  Author, A. A. (Year). Title [Doctoral dissertation, University]. Repository. URL

Sitio web:
  Author. (Year). Title. Site. URL

Preprint:
  Author. (Year). Title. arXiv. https://arxiv.org/abs/XXXX
```

### Ejemplos APA
```
In-text:
  La inteligencia artificial ha transformado la educacion (Garcia, 2020, p. 45)
  [Cita parafraseada]

  "El aprendizaje profundo requiere grandes volumenes de datos etiquetados"
  (Lopez & Martinez, 2021, p. 12) [Cita textual]

Referencia:
  Garcia, R. (2020). Inteligencia artificial en educacion. Revista de Innovacion, 15(2), 40-55. https://doi.org/10.1234/abcd
```

## IEEE

### Configuracion del documento
| Elemento | Valor |
|----------|-------|
| Fuente | Times New Roman 10pt |
| Interlineado | Sencillo (1.0), doble entre secciones |
| Margenes | 2.54 cm (1 in) en los 4 lados |
| Columnas | 2 para el texto principal |
| Titulo | Centrado, 24pt, negrita |
| Encabezados de seccion | 10pt, mayusculas |
| Numeracion de paginas | Inferior centrado |
| Referencias | Numeradas [1], [2]... en orden de aparicion |

### Citas in-text
```
Como se demuestra en [1]...                  → referencia general
"...texto citado..." [2, p. 5]              → con pagina
Varios estudios [3], [4], [5] confirman...  → multiples
Como se menciona en [1, pp. 10-15]...       → rango de paginas
```

### Referencias
```
Articulo journal:
  [1] A. A. Author and B. B. Author, "Title," Journal Name, vol. X, no. Y, pp. Z-Z, Year. doi: XXXX.

Libro:
  [2] A. A. Author, Title (ed.). Publisher, Year.

Conference:
  [3] A. A. Author, "Title," in Proc. Conference Abbreviation, Year, pp. X-X.

Reporte tecnico:
  [4] A. A. Author, "Title," Institution, Tech. Rep. X, Year.

Estandar:
  [5] Standard Name, Standard ID, Year. [Online]. Available: URL

Sitio web:
  [6] A. A. Author, "Title," Site, Year. [Online]. Available: URL
```

### Ejemplos IEEE
```
In-text:
  El metodo propuesto mejora la precision en un 15% [1, p. 234]
  [Cita parafraseada]

  "Los algoritmos geneticos permiten optimizar funciones no lineales"
  [2, p. 89] [Cita textual]

Referencia:
  [1] M. Rodriguez and P. Sanchez, "Deep learning for medical imaging,"
  IEEE Trans. Med. Imaging, vol. 41, no. 5, pp. 1123-1135, 2024,
  doi: 10.1109/TMI.2024.1234567.
```

## Vancouver

### Configuracion del documento
| Elemento | Valor |
|----------|-------|
| Fuente | Times New Roman 12pt |
| Interlineado | Doble (2.0) |
| Margenes | 2.54 cm (1 in) en los 4 lados |
| Alineacion | Izquierda |
| Numeracion de paginas | Superior derecha |
| Referencias | Numeradas en orden de aparicion |

### Citas in-text
```
Segun estudios previos (1,2)...             → (numero)
"...cita textual..." (3).                   → al final
Varios autores (4-6) coinciden...           → rango
```

### Referencias
```
Articulo journal:
  1. Author AA, Author BB. Title. Abbreviated Journal Name. Year;Volume(Issue):Pages. doi:XXXX

Libro:
  2. Author AA. Title. Edition. Place: Publisher; Year.

Capitulo de libro:
  3. Author AA. Chapter title. In: Editor AA, editor. Book title. Place: Publisher; Year. p. X-X.

Conference:
  4. Author AA. Title. In: Proceedings of Conference Name; Date; Location. Place: Publisher; Year. p. X-X.

Tesis:
  5. Author AA. Title [dissertation]. Place: University; Year.

Sitio web:
  6. Author AA. Title [Internet]. Place: Publisher; Year [cited YYYY MMM DD]. Available from: URL
```

### Ejemplos Vancouver
```
In-text:
  La prevalencia de diabetes tipo 2 aumenta globalmente (1,2) [Cita parafraseada]

  "La actividad fisica reduce el riesgo cardiovascular en un 30%"
  (3, p. 45) [Cita textual]

Referencia:
  1. Martinez L, Gomez R. Epidemiologia de la diabetes en America Latina.
  Rev Salud Publica. 2023;45(2):123-35. doi:10.1590/2023.12345
```

## Reglas Generales Aplicables

Cada norma existe para resolver un problema distinto: APA prioriza la trazabilidad autor-fecha (util en ciencias sociales donde la actualidad del conocimiento importa), IEEE usa numeracion secuencial (eficiente en ingenieria donde se citan muchos estandares y patentes), Vancouver tambien usa numeracion pero con formato distinto (optimizado para revistas medicas). Elegir la norma correcta no es burocracia: es lo que permite a tus lectores encontrar las fuentes sin friction.

| Elemento | APA 7th | IEEE | Vancouver |
|----------|---------|------|-----------|
| Orden referencias | Alfabetico por apellido | Orden de aparicion [1], [2]... | Orden de aparicion (1), (2)... |
| Enlace en referencia | DOI/URL | DOI/URL | DOI/URL |
| Formato autor | Apellido, Inicial. | Inicial. Apellido | Apellido Inicial |
| & vs "y" | & entre parentesis, "y" en texto | N/A | N/A |
| Enaltecer titulo | Cursiva en libro/journal | Comillas en articulo | Cursiva en libro/journal |
| Volumen | Cursiva | Negrita | Sin formato especial |
| DOI | https://doi.org/XXXX | doi: XXXX | doi:XXXX |

---

## Restricciones
- NO mezclar formatos de diferentes normas en un mismo documento
- NO omitir el DOI cuando esta disponible
- NO usar "Recuperado de" en APA 7th (se usa la URL/DOI directamente)
- NO incluir referencias sin enlace activo verificable
